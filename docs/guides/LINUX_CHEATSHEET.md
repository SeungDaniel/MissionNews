# 🐧 리눅스/맥 터미널 필수 명령어 모음 (Linux Cheatsheet)

이 문서는 서버 배포 및 파일 관리 시 자주 사용하는 핵심 명령어들을 정리한 파일입니다. 
필요에 따라 계속 업데이트됩니다.

---

## 📁 1. 폴더 생성 (Directory)

| 명령어 | 설명 | 예시 |
| :--- | :--- | :--- |
| `mkdir [이름]` | 폴더 하나를 생성합니다. (Make Directory) | `mkdir data` |
| **`mkdir -p [경로]`** | **(추천)** 상위 폴더가 없으면 그것까지 한 번에 다 만듭니다. | `mkdir -p data/temp/2024` |

---

## 🚚 2. 이동 및 이름 변경 (Move/Rename)

리눅스에서는 '이동'과 '이름 변경'이 같은 명령어(`mv`)입니다.

| 명령어 | 설명 | 예시 |
| :--- | :--- | :--- |
| `mv [파일] [폴더]` | 파일을 해당 폴더로 **이동**합니다. | `mv main.py scripts/` |
| `mv [옛이름] [새이름]` | 파일/폴더의 **이름을 변경**합니다. | `mv README.md README_OLD.md` |
| `mv [폴더1] [폴더2]` | 폴더1을 폴더2 안으로 이동시킵니다. | `mv src/ old_src/` |
| `mv *.py [폴더]` | 확장자가 `.py`인 모든 파일을 이동합니다. (와일드카드) | `mv *.py src/` |

---

## 🗑️ 3. 삭제 (Remove) - ⚠️ 주의! 복구 불가 ⚠️

**삭제 명령어는 휴지통으로 가지 않고 즉시 지워지므로 매우 신중해야 합니다.**

| 명령어 | 설명 | 예시 |
| :--- | :--- | :--- |
| `rm [파일]` | 파일 하나를 삭제합니다. | `rm unused.py` |
| `rmdir [폴더]` | **빈 폴더**만 삭제합니다. (안에 파일 있으면 에러) | `rmdir empty_folder` |
| **`rm -rf [폴더]`** | **(강력)** 폴더와 그 안의 모든 내용을 강제로 삭제합니다. | `rm -rf nginx` |

> *Tip: `rm -rf`를 쓸 때는 엔터 치기 전에 꼭 경로를 한 번 더 확인하세요!*

---

## 👀 4. 확인 및 탐색 (List/Nav)

| 명령어 | 설명 | 예시 |
| :--- | :--- | :--- |
| `pwd` | 현재 내가 있는 경로(위치)를 보여줍니다. | `pwd` → `/home/user/app` |
| `ls` | 현재 폴더의 파일 목록을 간단히 봅니다. | `ls` |
| **`ls -al`** | **(추천)** 숨겨진 파일(.env 등) 포함, 자세한 정보를 봅니다. | `ls -al` |
| `cd [폴더]` | 해당 폴더로 들어갑니다. (Change Directory) | `cd config` |
| `cd ..` | 상위 폴더(부모 폴더)로 나갑니다. | `cd ..` |

---

## ⌨️ 5. 유용한 꿀팁 (Tips)

1.  **자동 완성 (`Tab` 키)**
    *   파일 이름이나 폴더 이름을 칠 때, 앞글자 한두 개만 치고 **`Tab`** 키를 누르면 자동으로 완성됩니다. 오타를 줄이는 최고의 습관입니다.
2.  **명령어 기록 (`Arrow Up` 키)**
    *   **방향키 위(↑)**를 누르면 조금 전 입력했던 명령어들이 나옵니다. 똑같은 명령어를 다시 칠 필요가 없습니다.
3.  **화면 지우기**
    *   `clear` 명령어를 치면 터미널 화면이 깨끗해집니다.
