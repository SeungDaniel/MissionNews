## 2. 체크리스트 형식 (상세)

### 1. 간증 영상 다운로드

**필요 도구:** 방송실 나스 서버


[유튜브] 권신찬 https://studio.youtube.com/channel/UCQSQJ5tJfzVXprhvoajyelg?c=UCQSQJ5tJfzVXprhvoajyelg

[시트] 간증 관리 https://docs.google.com/spreadsheets/d/1jNLwoIMMwaGf04BQ_f6Gwg2VXY3x-1J1u0OqTlTdZeQ/edit?gid=0#gid=0

[서버] 방송실 https://anseongbc.synology.me:5001/fsdownload/Qd9RLnq54/%ED%95%B4%EC%99%B8%EA%B0%84%EC%A6%9D

[서버] 인터넷 https://archive-storage.quickconnect.to

- [ ] (1) 방송실 나스 서버 접속
- [ ] (2) 해당 날짜 폴더로 이동하여 간증 영상 다운로드

### 2. 간증 정보 등록

**필요 도구:** 간증 관리 시트 

[시트] 간증 관리 https://docs.google.com/spreadsheets/d/1jNLwoIMMwaGf04BQ_f6Gwg2VXY3x-1J1u0OqTlTdZeQ/edit?gid=0#gid=0

- [ ] (1) 간증 관리 시트 접속
- [ ] (2) 간증 영상 정보 입력

### 3. 파일명 변경

- [ ] 간증 영상 파일명 변경 (별도 세부 절차 없음)

### 4. 유튜브 업로드

**필요 도구:** 권신찬 유튜브 스튜디오

[유튜브] 권신찬 https://studio.youtube.com/channel/UCQSQJ5tJfzVXprhvoajyelg?c=UCQSQJ5tJfzVXprhvoajyelg

- [ ] (1) 권신찬 유튜브 관리자 페이지 접속
- [ ] (2) 업로드할 간증 영상 파일 선택하여 업로드

### 5. 썸네일 이미지 작업

**필요 도구:** 영상 플레이어, 이미지 편집 툴

- [ ] (1) 간증 영상을 재생하여 썸네일 이미지 캡쳐
- [ ] (2) 캡쳐한 이미지 4:3 규격으로 수정
- [ ] (3) 간증 썸네일 이미지 파일명을 간증 영상 파일명과 동일하게 변경

### 6. 간증 영상 서버 업로드

**필요 도구:** 내부 서버

[서버] 인터넷 https://archive-storage.quickconnect.to

- [ ] (1) 위 경로의 ‘신규’ 폴더에 영상 파일과 썸네일 이미지 파일 업로드
- [ ] (2) ‘신규’ 폴더의 간증 영상의 AI 학습이 완료되면, 폴더 이동

### 7. 권신찬 유튜브 영상 공개 준비

**필요 도구:** 권신찬 유튜브 스튜디오

[유튜브] 권신찬 https://studio.youtube.com/channel/UCQSQJ5tJfzVXprhvoajyelg?c=UCQSQJ5tJfzVXprhvoajyelg

- [ ] (1) 간증 소개글 작성
- [ ] (2) 유튜브 영상 정보 업데이트

### 8. 텔레그램 공유

**필요 도구:** 간증 관리 시트, 텔레그램 (EBC 공지사항, 새포럼 공지방, 선교소식)

[시트] 간증 관리 https://docs.google.com/spreadsheets/d/1jNLwoIMMwaGf04BQ_f6Gwg2VXY3x-1J1u0OqTlTdZeQ/edit?gid=0#gid=0

- [ ] (1) 간증 관리 시트에 등록한 간증들은 미리 업로드 일정 잡기
- [ ] (2) 텔레그램 ‘EBC 공지사항’ 채널에 간증 업로드
- [ ] (3) 텔레그램 ‘새포럼 공지방’과 ‘선교소식’ 그룹에 전달 예약하기
- [ ] (4) 텔레그램 ‘새포럼 공지방’과 ‘선교소식’ 그룹에 예약이 잘 되었는지 확인

### 9. 지난 간증 삭제

**필요 도구:** 텔레그램 (새 포럼 대화방)

- [ ] 텔레그램 ‘새 포럼 대화방’ 그룹에 게시된 지난 간증 삭제
